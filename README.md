# Compace

**Democratizing access to academic competitions for students globally.**

Compace is a full-stack platform that helps students discover, track, and prepare for high-value opportunities. It solves the "discovery gap" by aggregating data and providing AI-driven preparation tools.

## Tech Stack
- **Frontend:** Next.js (React), Tailwind CSS, TypeScript
- **Backend:** Node.js, Prisma ORM
- **AI Engine:** Google Gemini API (for generating study plans)
- **Data Pipeline:** Python (BeautifulSoup/Pandas) for ETL

## Status
In active development.
